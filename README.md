# Bark Out

## 概要
Bark Outはブロック崩しをベースに制作したアクションゲームです。  
パドルを操作してボールを反射させながらブロックを破壊していきます。  
教材として配布されたブロック崩しゲームをベースに、弾による攻撃機能やギャラリー収集要素を追加しました。

## 使用技術
- C++
- DX ライブラリ
- Visual Studio

操作方法
- パドル移動：矢印キー

## 実装した機能
- パドル操作
- ボール反射
- ブロック破壊
- スコア管理
- ステージ制
- ギャラリー機能
- シーン遷移
